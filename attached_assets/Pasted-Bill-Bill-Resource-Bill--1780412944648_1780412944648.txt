Order Item Resource
Структура объекта товара (используется в массиве "items" при создании счёта)
Важно: передача пустой строки ("") или строки "NULL" в любом поле является недопустимой. Если данные для поля отсутствуют — поле необходимо полностью исключить из запроса.
Название параметра	Тип	Формат	Возможные значения	Пример	Описание
name	string			Steam	Наименование товара или услуги
price	decimal			500.00	Стоимость позиции в виде строки с двумя знаками после запятой (например: 500.00)
quantity	decimal			1	Количество единиц товара в виде строки (например: 1)
category	string			steam	Внутренняя категория продукта. Примеры: steam, digital/stars, digital/subscription/xbox, digital/subscription/netflix
extra	array			{"steam_account": "jordas123"}	Объект с идентификатором аккаунта, связанным с покупкой. Набор обязательных полей зависит от категории товара. Если данных нет — поле необходимо полностью исключить из запроса, не передавать пустые или нулевые значения.
extra[steam_account]	string			jordas123	Логин Steam-аккаунта, на который производится пополнение. Обязателен для товаров категории Steam.
extra[telegram_username]	string			DurovunaDR	Username Telegram-аккаунта для пополнения Stars (например: DurovunaDR). Обязателен при продаже через веб-сайт и через Telegram-бот.
extra[telegram_id]	string			5099904440	Числовой идентификатор Telegram-аккаунта (например: 5099904440). Обязателен только при продаже через Telegram-бот.
extra[account]	string			user@example.com	Аккаунт или email, на который активируется услуга (например: user@example.com). Передавать только при наличии данных.

POST /api/v1/bill/create
Создать счет на оплату
Для того чтобы провести оплату необходимо создать счет. Если тип счета многоразовый, то по одному счету можно произвести несколько платежей. Домен в shop_url должен совпадать с доментом success_url и fail_url.
Параметры запроса
Название параметра	Обязательный параметр	Тип	Возможные значения	Пример	Описание
amount	Да	decimal		380.99	Сумма счета на оплату
shop_id	Да	string		LXZv3R7Q8B	Уникальный идентификатор магазина, к которому относится платеж. Без этого параметра не будет работать Success URL, Fail URL и Result URL
order_id	Нет	string		order-285394168	Уникальный идентификатор заказа. Будет возвращен в postback.
description	Нет	string		Order #285394168	Описание платежа
type	Нет	enum	normal
multi	normal	Тип платежа. Одноразовый или многоразовый. Если выбран одноразовый, то второй раз оплатить не получится.
locale	Нет	enum	en
ru	en	Локаль, язык в котором будет отображаться форма платежа
currency_in	Нет	enum	RUB
USD
EUR	RUB	Валюта, в которой оплачивается счет. Если не передана, то используется валюта магазина. Если shop_id не определен, то используется RUB.
custom	Нет	string		my-custom-string	Произвольное поле. Будет возвращено в postback.
payer_pays_commission	Нет	enum	1
0	1	Параметр, который указывает на то, кто будет оплачивать комиссию за входящий платёж.
payer_data[phone]	Нет	string		+79991234567	Номер телефона плательщика.
payer_data[email]	Нет	string		payer@email.com	Параметр, который заполняет email клиента на платёжной странице.
name	Нет	string		Donation	Название ссылки. Укажите, за что принимаете средства. Этот текст будет отображен в платежной форме.
ttl	Нет	integer		600	Время жизни счета на оплату в секундах.
return_url	Нет	string		https://exmpl.com/return	URL для кнопки "Назад в магазин" на странице оплаты. Домен должен совпадать с тем, что указан при добавлении магазина в систему
success_url	Нет	string		https://exmpl.com/order-285394168/success	Страница успешной оплаты.
fail_url	Нет	string		https://exmpl.com/order-285394168/fail	Страница неуспешной оплаты.
payment_method	Нет	enum	BANK_CARD
SBP	SBP	Если указан этот параметр, то при переходе на платежную форму этот способ оплаты будет выбран автоматически, без возможности выбора другого способ оплаты.
request_fields[email]	Нет	boolean		false	Обязательный запрос электронной почты у плательщика
request_fields[phone]	Нет	boolean		false	Обязательный запрос номера телефона у плательщика
request_fields[name]	Нет	boolean		false	Обязательный запрос ФИО у плательщика
request_fields[comment]	Нет	boolean		false	Обязательный запрос комментария у плательщика
items	Нет	array		Order Item Resource	Список товаров
Параметры ответа
Название параметра	Тип	Формат	Возможные значения	Пример	Описание
success	boolean		true
false	true	Флаг успешности запроса
link_url	string	URL		https://pally.info/link/5QWlqB2kKJ	Ссылка на страницу с QR кодом
link_page_url	string	URL		https://pally.info/transfer/5QWlqB2kKJ	Ссылка на оплату
bill_id	string			5QWlqB2kKJ	Уникальный идентификатор счета
Пример запроса
curl -X POST 'https://pal24.pro/api/v1/bill/create' \
-H 'Authorization: Bearer 72|oBCB7Z3SmUm1gvkpEdRcSR2q1ERHpG4vD3DNBmuT' \
-d 'amount=100.05' \
-d 'description=Some desc' \
-d 'order_id=15' \
-d 'type=multi' \
-d 'shop_id=LXZv3R7Q8B' \
-d 'custom=123' \
-d 'currency_in=RUB'
