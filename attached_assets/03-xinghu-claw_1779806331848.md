# Xinghu Claw — деплой OpenClaw за 1 минуту

Xinghu Claw — десктопный инструмент для развёртывания OpenClaw AI-бота **за 1 минуту** в Telegram, Feishu, DingTalk, Discord и других платформах. Не нужны технические знания, сервер или ручная настройка окружения.

🌐 Официальный сайт: [xinghuclaw.com](https://xinghuclaw.com/)

---

## Ключевые возможности

- Автоматическая установка Node.js, Git и зависимостей
- Поддержка Claude, GPT, Gemini — переключение в один клик
- Поддержка Telegram, Feishu, DingTalk, Discord
- Визуальная панель управления всеми экземплярами ботов

---

## Поддерживаемые ОС

| ОС | Требование |
|---|---|
| Windows 11 | 64-битная система |
| macOS | Apple Silicon (M1/M2/M3/M4) |

---

## Предварительные требования

- API-ключ Xinghu (начинается с `sk-`) из раздела **Управление токенами**
- Токен бота нужной платформы (Telegram Bot Token и т.д.)

---

## Установка

### Windows
1. Скачайте `.exe`-установщик с [xinghuclaw.com](https://xinghuclaw.com/)
2. Запустите `openclaw-manager-vXX.exe`
3. При предупреждении безопасности: **«Подробнее» → «Всё равно запустить»**

### macOS
1. Скачайте `.pkg`-файл (вкладка macOS)
2. Установите, открыв `.pkg`
3. При блокировке: **Системные настройки → Конфиденциальность → «Всё равно открыть»**

---

## Шаг 1: Инициализация окружения

При первом запуске программа автоматически установит Node.js (v22+), Git и OpenClaw CLI.  
Дождитесь, пока все пункты станут ✅.

---

## Шаг 2: Создание экземпляра бота

Нажмите **«Новый экземпляр»** и следуйте мастеру:

1. **Имя** — латинские буквы, цифры, `_` и `-`
2. **Модель** — Claude Opus/Sonnet/Haiku, GPT-5, Gemini 3.0/3.1 + API-ключ Xinghu
3. **Платформа** — Telegram, Feishu, Discord, DingTalk или только API + токен бота

---

## Шаг 3: Деплой

Нажмите **«Деплой»** — программа автоматически выполнит установку, настройку и запуск шлюза. По завершении бот онлайн ✅

## Управление ботом

| Действие | Как выполнить |
|---|---|
| Запуск / остановка | Кнопка «Запустить» / «Остановить» |
| Смена модели | «Сменить модель» — вступает в силу немедленно |
| Обновление ключа | «Обновить API Key» |
| Потоковые ответы (Telegram) | Включить функцию «Стриминг» |
| Просмотр логов | Кнопка «Логи» |
| Восстановление при сбоях | Кнопка «Исправить» |

## Частые вопросы

**Браузер не открылся автоматически?**  
Откройте вручную: `http://127.0.0.1:51942`

**macOS: «Невозможно открыть»?**  
Системные настройки → Конфиденциальность → «Всё равно открыть»

**Зависло на инициализации?**  
Проверьте подключение к интернету.

**Где хранятся данные?**  
- Windows: `%APPDATA%\openclaw-manager\profiles\`
- macOS: `~/Library/Application Support/openclaw-manager/profiles/`

**Можно запускать несколько ботов?**  
Да, каждый экземпляр независим.
