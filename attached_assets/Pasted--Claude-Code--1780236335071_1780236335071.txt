📌Claude Code — это мощный помощник по программированию ИИ, который позволяет работать с ИИ непосредственно на вашем устройстве. Этот урок проведёт вас через процесс установки и настройки.
Node.js Версия ≥ 18.0
Поддерживаемые операционные системы: macOS, Linux, Windows (WSL)
1. Установить Node.js
📌Совет: если у вас установлен Node.js версии 18.0 или новее, вы можете пропустить этот шаг.
Пользователи Windows
Откройте официальный сайт node.js, чтобы скачать и установить
Пользователи Ubuntu / Debian
#安装 Node.js LTS 版本
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
Пользователи macOS
# 安装 Xcode 命令行工具
sudo xcode-select --install

# 安装 Homebrew（如果尚未安装）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 通过 Homebrew 安装 Node.js
brew install node

# 验证安装
node --version










Установка Git
Claude Code требует Git для управления кодом:
1.
Посетите https://git-scm.com, чтобы скачать и установить
2.

Если номер версии можно правильно вывести, это значит, что установка прошла успешно
Совет: когда пользователи Windows устанавливают Git, рекомендуется проверить опцию «Добавить Git в PATH», чтобы убедиться, что команда может использоваться на любом терминале.git










2. Установить Claude Code

3. Настройте и начните использовать его
После завершения кода Клода нам всё равно нужно получить необходимую информацию о конфигурации; иначе её нельзя запустить и использоватьЗайдите в https://apinet.cloud для регистрации и входа, затем перейдите на страницу Панель управления — > API Tokens, чтобы скопировать токен для резервного копирования.







Вам нужно подготовить два важных элемента конфигурации:

Элементы конфигурации

Используйте https://apinet.cloud

Как использовать код Клода в VScode
Метод 1: Установка переменных компьютерной системы
📌Потому что по умолчанию Claude Code требует входа в официальный аккаунт участника Claude или в API разработчика для его использования. Если вы хотите вызвать его через наш сторонний API Xinghu Cloud, сначала нужно настроить переменные пользовательской среды!
Шаг первый
Откройте окно настроек переменной среды на вашем компьютере
Всё равно добавляйте API токены и адреса сервисов API, как и раньше
变量名：ANTHROPIC_AUTH_TOKEN   变量值：sk-...  # 替换为您的星狐云令牌密钥
变量名：ANTHROPIC_BASE_URL   变量值：https://apinet.cloud
Шаг второй: Откройте VScode и установите плагин Claude Code
После установки иконка кода Клода появится в правом верхнем углу окна редактирования. Откройте её, и вы сможете начать пользоваться!
Метод 2: Напрямую настройте настройки в VSCode
Откройте настройки VSCode, введите 'claude' в поисковую строку (плагин должен быть установлен), затем нажмите на setting.json файл, отображаемый ниже
В переменной среды claudecode под JSON-файлом добавьте пример кода в диаграмму и измените ключ к своему созданию

{
      "name": "ANTHROPIC_BASE_URL",
      "value": "https://apinet.cloud"
    },
    {
      "name": "ANTHROPIC_AUTH_TOKEN",
      "value": "星狐云令牌密钥"
    }
❓ Часто задаваемые вопросы
Вопрос: Столкновение с «Invalid API Key · Пожалуйста, запустите ошибку /login"?
A: Это указывает на то, что Claude Code не обнаружил переменные среды. Пожалуйста, проверьте:
Правильно ли установлены ANTHROPIC_AUTH_TOKEN и ANTHROPIC_BASE_URL
Правильно ли значение переменной среды (токены начинаются с sk-)
Если используется постоянная конфигурация, был ли терминал перезапущен?
Вопрос: Почему отображается статус «офлайн»?
Ответ: Код Клода определяет статус сети через подключение к Google. Отображение «офлайн» не влияет на обычное использование, это просто означает, что Google не может быть подключен.Вопрос: Почему Fetch не работает при просмотре веб-страниц?
Ответ: Claude Code должен позвонить в службу Claude для проверки безопасности перед тем, как перейти к веб-странице. Вам понадобятся:
Поддерживайте стабильное международное интернет-соединение
При необходимости используйте глобальные прокси
Вопрос: Всегда ли запрос показывает «fetch failed»?
О: Проблема может быть вызвана сетевой средой. Решение:Попробуйте использовать прокси-инструменты
Вопрос: Как мне обрабатывать ошибки API?
О: Это может быть вызвано нестабильными пересылающими агентами. Рекомендации:
Exit Claude Code (Ctrl+C)
Повторите команду клода
Если проблема сохраняется, попробуйте позже снова
Вопрос: Ошибка входа в веб-страницу?
О: Попробуйте очистить файлы cookie на сайте, а затем войти снова.
