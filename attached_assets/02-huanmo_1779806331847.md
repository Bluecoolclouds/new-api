# AI-рисование Huanmo

> 💡 При использовании модели Nano Banana Pro возможны редкие сбои генерации — это синхронизировано с серверами поставщика.

Сайт: [huanmotion.com](https://huanmotion.com)

---

## Настройка API

1. Перейдите на [huanmotion.com](https://huanmotion.com)
2. Нажмите **«Настройка API»** в правом верхнем углу
3. Нажмите **«Получить APIkey»**
4. Если токен Xinghu уже создан — введите его и сохраните

---

## Создание токена для рисования

1. Войдите на [xinghuapi.com](https://xinghuapi.com/)
2. Перейдите в **Управление токенами → Добавить токен**
3. Выберите группу: `优质gemini分组`, `优质gemini2分组` или `优质gemini3分组`
4. Вернитесь на Huanmo и введите ключ

---

## Пополнение баланса

Перейдите в [Кошелёк](https://xinghuapi.com/console/topup) и пополните:
- Онлайн-оплатой
- Кодом активации с Taobao
