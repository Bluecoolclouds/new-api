# Trae

Trae — мощная AI-среда разработки на базе VS Code с глубокой интеграцией AI. Поддерживает все плагины VS Code.

🌐 Скачать: [trae.cn](https://www.trae.cn/)

---

## Возможности Trae

- AI-чат и генерация кода прямо в IDE
- Умное автодополнение кода
- AI Agent — автоматическое выполнение сложных задач
- Полная совместимость с экосистемой плагинов VS Code

---

## Установка плагина Cline

Откройте Trae → **«Расширения»** → найдите `cline` → установите.

---

## Настройка Cline для Xinghu API

Нажмите значок Cline в боковой панели и заполните:

| Поле | Значение |
|---|---|
| Base URL | `https://xinghuapi.com/v1` |
| API Key | Создайте в разделе «Управление токенами» на Xinghu |
| Model | `claude-sonnet-4-5-20250929`, `gemini-2.5-pro` и др. |

Настройки сохраняются автоматически. Введите «Привет» для проверки ✅
