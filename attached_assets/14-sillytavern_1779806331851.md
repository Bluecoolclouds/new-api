# SillyTavern (Tavern AI)

SillyTavern — локальный интерфейс для ролевых игр и чата с AI.

📚 GitHub: [github.com/SillyTavern](https://github.com/SillyTavern/SillyTavern/blob/release/.github/readme-zh_cn.md)

---

## Возможности

- AI-ролевые игры с кастомными персонажами
- Поддержка голосового синтеза и динамических аватаров
- Совместное создание историй и сценариев
- TRPG-игры, интерактивная художественная литература

---

## Подключение Xinghu API

После запуска SillyTavern нажмите **«Подключить API»** в правом верхнем углу.

| Шаг | Поле | Значение |
|---|---|---|
| 1 | API | Завершение чата (Chat Completions) |
| 2 | Источник | Пользовательский (Custom) |
| 3 | URL | `https://xinghuapi.com/v1` |

**Шаг 4:** Нажмите **«Управление ключами» → «Добавить ключ API»** и введите токен Xinghu.

> 💡 Рекомендуемая группа: «Специальная группа для Tavern AI» или другая совместимая.

**Шаг 5:** В разделе **«Доступные модели»** выберите нужную и нажмите **«Подключить»**.
