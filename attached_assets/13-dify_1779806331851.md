# Dify

Dify — open-source платформа для создания LLM-приложений. Поддерживает сотни моделей, low-code разработку и приватное развёртывание.

- ⭐ 90k звёзд на GitHub
- 🏆 Входит в «Топ-100 глобальных AI-приложений»
- 🌐 Сайт: [dify.ai](https://dify.ai/zh)

---

## Сценарии использования

**Корпоративные:**
- AI-служба поддержки клиентов
- Автоматизация анализа данных

**Для разработчиков:**
- Быстрая разработка AI-приложений с визуальным интерфейсом

---

## Настройка

**1.** Войдите на [dify.ai](https://dify.ai/zh) → **Личный кабинет → Настройки**

**2.** Откройте **«Поставщики моделей»** и установите плагин **OpenAI-API-compatible**

**3.** Нажмите **«Добавить модель»** и заполните:

| Поле | Значение |
|---|---|
| Имя модели | Произвольное |
| API-ключ | Ваш токен Xinghu |
| URL | `https://xinghuapi.com` |

**4.** Нажмите **«Добавить»**

**5.** В рабочем пространстве → откройте приложение → в правом верхнем углу выберите добавленную модель
