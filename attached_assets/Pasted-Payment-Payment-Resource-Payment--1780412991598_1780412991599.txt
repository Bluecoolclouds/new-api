GET /api/v1/payment/search
Получить платежи
С помощью этого метода вы можете получить все платежи по магазину или за произвольный период времени.
Параметры запроса
Название параметра	Обязательный параметр	Тип	Возможные значения	Пример	Описание
start_date	Нет	datetime		2025-01-01	Начальная датавремя для получения платежей в UTC
finish_date	Нет	datetime		2024-12-01 23:59:59	Конечная датавремя для получения платежей в UTC
shop_id	Нет	string		LXZv3R7Q8B	Уникальный идентификатор магазина
per_page	Нет	integer		50	Количество элементов на странице
cursor	Нет	string		eyJpZCI6NzY1LC	Указатель на страницу
Параметры ответа
Название параметра	Тип	Формат	Возможные значения	Пример	Описание
success	boolean		true
false	true	Флаг успешности запроса
data	array	Payment		Cм. описание Payment Resource	Информация о платежах
links	array	Links		См. описание Pagination Links	Ссылки для пагинации
meta	array	Meta		См. описание Pagination Meta	Мета данные пагинации
Пример запроса
curl -X GET 'https://pal24.pro/api/v1/payment/search ?start_date=2024-01-01&finish_date=2025-12-01 ' \
-H 'Authorization: Bearer 72|oBCB7Z3SmUm1gvkpEdRcSR2q1ERHpG4vD3DNBmuT'

Возможные ошибки
Код ответа	Ключ ошибки	Описание ошибки
401	Unauthenticated	Неверный API Токен
403	api:error.access_denied	Мерчант не имеет доступ до магазина
403	api:error.ip_access_denied	IP не внесен в вайтлист
403	api:error.merchant_not_found	Мерчант не найден
403	api:error.bill_not_found	Счет не найден
403	api:error.shop_not_found	Магазин не найден. Возникает только в случае, если shop_id был передан
422		Ошибка валидации входных данных
