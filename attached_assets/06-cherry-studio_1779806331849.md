# Cherry Studio

Cherry Studio — open-source десктопный AI-клиент с поддержкой множества провайдеров.

🌐 Сайт: [cherry-ai.com](https://www.cherry-ai.com/)

---

## Шаг 1: Получить API-ключ

1. Войдите на [xinghuapi.com](https://xinghuapi.com/)
2. Перейдите в **Управление токенами → Добавить токен**
3. Скопируйте ключ формата `sk-...`

---

## Шаг 2: Открыть настройки Cherry Studio

Запустите Cherry Studio → значок **«Настройки»** → **«Поставщики моделей»**

---

## Шаг 3: Добавить поставщика

1. Нажмите **«Добавить поставщика»** или **«+»**
2. Имя: произвольное (например «Xinghu API»)
3. Тип: **OpenAI**
4. Заполните поля:

| Поле | Значение |
|---|---|
| URL | `https://xinghuapi.com` |
| API-ключ | Ваш токен Xinghu |

5. Добавьте модели вручную: `gpt-5`, `claude-sonnet-4-20250514`, `gemini-2.5-pro` и др.

---

## Шаг 4: Тест и сохранение

Нажмите **«Проверить соединение»** → при успехе нажмите **«Сохранить»**.

---

## Шаг 5: Начало работы

**Главный экран → Добавить ассистента → Новый диалог → выбрать модель из «Xinghu API»**
