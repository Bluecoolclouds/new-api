# N8N — Автоматизация с Xinghu API

---

## Рабочий процесс AI-чата

**1.** Создайте новый рабочий процесс в N8N.

**2.** Выберите триггер (например «Ручной запуск»).

**3.** Добавьте узел **HTTP Request** и настройте:

**Метод:** `POST`

**URL:**
```
https://xinghuapi.com/v1/chat/completions
```

**Заголовки:**
```json
{
  "Authorization": "Bearer ВАШ_КЛЮЧ_XINGHU",
  "Content-Type": "application/json"
}
```

**Тело запроса:**
```json
{
  "model": "gemini-2.5-pro-all",
  "messages": [
    { "role": "system", "content": "Вы — AI-ассистент" },
    { "role": "user", "content": "Привет" }
  ]
}
```

**4.** Нажмите **«Execute step»** — в разделе OUTPUT появится ответ модели.

## Рабочий процесс генерации изображений

**URL:**
```
https://xinghuapi.com/v1/images/generations
```

**Тело запроса:**
```json
{
  "model": "gemini-3-pro-image-preview",
  "prompt": "Ваш запрос",
  "size": "1024x1024",
  "n": 1,
  "response_format": "url"
}
```
