# Cursor

Cursor — AI-редактор кода на базе VS Code.

🌐 Сайт: [cursor.com](https://cursor.com/cn)

---

## Настройка через плагин Cline

**Шаг 1:** Откройте маркетплейс Cursor, найдите **Cline** и установите.

**Шаг 2:** Значок Cline появится в верхнем меню. Откройте его.

**Шаг 3:** Нажмите значок **настроек** в правом верхнем углу.

**Шаг 4:** Заполните параметры:

| Поле | Значение |
|---|---|
| API Provider | OpenAI Compatible |
| Base URL | `https://xinghuapi.com/v1` |
| API Key | Ваш токен Xinghu |
| Model | Например `claude-sonnet-4-6` |

**Шаг 5:** Для проверки введите в чат Cline «Привет» — если получите ответ, настройка выполнена ✅
