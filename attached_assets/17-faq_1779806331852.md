# Часто задаваемые вопросы (FAQ)

---

## HTTP-статусы и их значения

| Статус | Описание | Что делать |
|---|---|---|
| `400 Bad Request` | Неверный формат запроса | Проверьте структуру JSON-тела запроса |
| `401 Unauthorized` | Ошибка проверки API-ключа | Убедитесь в правильности ключа и его сроке действия |
| `403 Forbidden` | Недостаточно прав доступа | Проверьте группу токена и доступные модели |
| `404 Not Found` | Ресурс не найден | Проверьте URL и название эндпоинта |
| `413 Request Entity Too Large` | Слишком большое тело запроса | Уменьшите объём передаваемых данных |
| `429 Too Many Requests` | Превышен лимит запросов | Проверьте баланс, снизьте частоту запросов |
| `500 Internal Server Error` | Внутренняя ошибка сервера | Повторите запрос позже |
| `503 Service Unavailable` | Сервер временно недоступен | Ведётся техническое обслуживание |

## Общие вопросы

**Почему модель не отвечает?**  
Скорее всего, выбрана неподходящая группа токена. Создайте новый токен с нужной группой.

**Можно ли использовать один ключ для всех приложений?**  
Да, один токен Xinghu работает со всеми поддерживаемыми приложениями.

**Что делать, если баланс закончился?**  
Пополните кошелёк на [xinghuapi.com/console/topup](https://xinghuapi.com/console/topup).

**Как проверить расход токенов?**  
В панели управления Xinghu есть раздел с историей запросов и расходом баланса.

**Поддерживается ли потоковая передача (streaming)?**  
Да, большинство моделей поддерживают `stream: true` в запросе.
